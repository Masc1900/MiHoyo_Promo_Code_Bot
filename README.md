# MiHoyo_Promo_Code_Bot

Un bot Discord che manda su un canale di testo un messaggio contenente il promo code più recente per i giochi MiHoyo:

- Genshin Impact
- Honkai Star Rail
- Zenless Zone Zero

prende i codici da Game8 usando [questo strumento](https://github.com/Masc1900/MiHoyo_Codes_Scraper) che converte i codici delle pagine in un file Json.
